# Design Pipeline Skill — 이서용 (매니저)

## 🏆 추천 조합 (Golden Combinations)

### A. 브랜딩/로고 (깔끔함, 텍스트 중요)
```
생성: 나노바나나 Pro (텍스트 렌더링 우수) 또는 Ideogram (텍스트 특화)
후가공: Vectorizer.ai (PNG→SVG) + Remove.bg (배경 제거)
검수: Gemini Vision (가독성/단순성 평가)
```

### B. UI/UX 에셋 (일관성 중요)
```
생성: 나노바나나 (빠른 시안) → 나노바나나 Pro (최종)
색상: design-system 스킬의 토큰 참조
검수: Gemini Vision (디자인 시스템 일관성 평가)
연동: HTML 프로토타입 직접 반영
```

### C. 마케팅/광고 이미지 (디테일 중요)
```
생성: 나노바나나 Pro (4K 고해상도)
후가공: 업스케일링 (Phase 3에서 도입)
검수: Gemini Vision (품질 평가 + 광고 적합성)
```

## 핵심 프로세스: Vision + Generation 피드백 루프

**이것이 퀄리티를 결정하는 가장 중요한 프로세스.**

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  1. 생성     │────▶│  2. 비평     │────▶│  3. 수정     │
│  (Generate)  │     │  (Critique)  │     │  (Refine)    │
│  나노바나나   │     │  Gemini Vision│    │  프롬프트수정  │
└─────────────┘     └─────────────┘     └──────┬──────┘
                                                │
                         ┌──────────────────────┘
                         ▼
                    ┌─────────────┐     ┌─────────────┐
                    │  4. 재생성   │────▶│  5. 최종     │
                    │  (Re-roll)   │     │  (Finalize)  │
                    │  개선된프롬프트│     │  저장/배포    │
                    └─────────────┘     └─────────────┘
```

### 비평 프롬프트 템플릿
```
이 이미지를 전문 디자이너 관점에서 평가해줘:

1. 구도/레이아웃: 균형감, 여백, 시선 흐름
2. 색상: 조화, 대비, 브랜드 일관성
3. 타이포그래피: 가독성, 철자, 크기 적절성
4. 기술적 품질: 해상도, 아티팩트, 경계 선명도
5. 목적 적합성: [앱 아이콘/로고/배너/마케팅] 용도에 적합한가?

점수 (1-10): 
7점 이하면 구체적 개선 사항 제시.
```

### 최대 반복 횟수: 3회
- 3회 반복 후에도 7점 이하면 프롬프트 자체를 재설계

## 에셋 유형별 프로세스

### 앱 아이콘
1. 나노바나나(일반)로 시안 3개 생성 → ~$0.09
2. Gemini Vision으로 3개 비교 평가 → 최적 1개 선택
3. 나노바나나 Pro로 최종 생성 (1:1, 2K) → ~$0.13
4. 피드백 루프 1-2회 → ~$0.26
5. (선택) Vectorizer.ai로 SVG 변환
6. **총 비용: ~$0.50 이하**

### 로고
1. 나노바나나 Pro로 텍스트 포함 시안 생성 → ~$0.13
2. Gemini Vision으로 가독성/단순성 평가
3. 피드백 루프 2-3회 → ~$0.39
4. Vectorizer.ai로 SVG 변환 (필수)
5. Remove.bg로 배경 제거
6. **총 비용: ~$0.60 + 후가공 비용**

### 마케팅 배너 (앱스토어 스크린샷 배경)
1. 나노바나나 Pro 4K로 생성 → ~$0.24
2. Gemini Vision으로 품질 평가
3. 피드백 루프 1-2회 → ~$0.48
4. **총 비용: ~$0.80 이하**
